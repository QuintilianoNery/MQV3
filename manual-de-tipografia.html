Sobre este documento

A tipografia é a voz visual da MQV³. Este manual define o sistema tipográfico completo da marca — quais fontes usar, em quais tamanhos, pesos e espaçamentos, para cada contexto de aplicação. O sistema é construído em três camadas complementares: Display para identidade e impacto, Texto para leitura e comunicação, e Mono para dados, códigos e especificações técnicas.

Regras de uso — Pode / Não pode

✕ Não fazer

Usar Orbitron para texto corrido ou parágrafos longos. Sua legibilidade em tamanho de corpo é limitada — ela existe para impacto, não para leitura.

Ex. errado: Corpo de e-mail em Orbitron 12px → ilegível, sobrecarregado.

✓ Correto

Reservar Orbitron exclusivamente para wordmark, títulos principais (H1/H2), KPIs em destaque e elementos de identidade com no mínimo 13–14px/pt.

Ex. certo: Título de seção em Orbitron 700 · 20px + corpo em Sora 400 · 14px.

✕ Não fazer

Misturar mais de três fontes num mesmo documento ou peça de comunicação. O sistema tem três fontes exatas — usar apenas elas.

Ex. errado: Orbitron + Sora + JetBrains + Montserrat + Georgia → caos visual.

✓ Correto

Usar somente Orbitron, Sora e JetBrains Mono. Em caso de indisponibilidade de webfont, usar os fallbacks declarados no manual (Courier New / system-ui).

Ex. certo: Sora 700 · heading + Sora 400 · corpo + JetBrains · metadata.

✕ Não fazer

Aplicar texto em Teal Neon (#0FDCC0) sobre fundos claros (Névoa ou branco) em tamanho de corpo. Não atinge contraste mínimo WCAG.

Ex. errado: Parágrafo em Teal 12px sobre #F3F7FB → contraste 2.1:1 (reprovado).

✓ Correto

Usar Teal Neon em texto apenas sobre fundo escuro (Midnight ou Cobalto), onde atinge contraste AAA. Em fundos claros, usar Índigo ou Cobalto para texto.

Ex. certo: Label em Teal 9px Caps sobre Midnight → contraste 8.9:1 (AAA).

✕ Não fazer

Usar peso 100 ou 200 da Sora para texto informativo. Pesos muito leves perdem legibilidade em tamanhos menores que 14px e especialmente em impressão.

✓ Correto

Peso mínimo recomendado para texto é Sora 300 em tamanhos ≥ 12px (digital) ou ≥ 8pt (impressão). Para telas pequenas e mobile, preferir Sora 400.
